# Slotted Media — Supply Chain Software Suite

10 tools for warehouse and supply chain operations, built with React.

## Quick Start (5 minutes)

### Prerequisites — install these first:
1. **Node.js** — download from https://nodejs.org (pick LTS version)
2. **Git** — download from https://git-scm.com

### Run locally:
```bash
npm install
npm run dev
```
Open http://localhost:5173 in your browser.

### Deploy to Vercel:
```bash
git init
git add .
git commit -m "initial commit"
```
Then push to GitHub and import in Vercel. See the Migration Guide document for detailed steps.
